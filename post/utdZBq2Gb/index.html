如何利用you-get包下载网页上的视频

2019-10-26

大家在社团工作或者日常生活中经常看到比较好看的视频，想要下载下来，但是有的网页并不提供下载，那么如何方便快捷的下载网页中出现的视频呢，这里提供python中的一个包you-get（youtube-dl可以获得更多下载选项）来演示。
首先要确保你的电脑已经安装了python3.7，并且已经添加了环境变量，检验方法：win+R，输入cmd，打开命令提示符，输入python，这时会显示：

然后利用pip安装you-get包，在命令行输入pip install you-get

安装成功之后，我们首先打开浏览器，找到你想下载的视频播放界面，复制链接，图中链接为
https://www.bilibili.com/video/av71572166

在命令行输入you-get https://www.bilibili.com/video/av71572166，开始下载，

下载完成，图中xml为弹幕文件。

缺点是不能选择清晰度自定义下载，如果想选择清晰度的话，需要用到另一个包youtube-dl，有空的话再写。
